### Powerline fonts

- [powerline/fonts: Patched fonts for Powerline users.](https://github.com/powerline/fonts)

```sh
# clone
git clone https://github.com/powerline/fonts.git --depth=1

# install
cd fonts
./install.sh

# clean-up a bit
cd ..
rm -rf fonts
```
